# Изображения для баннеров

В этой папке должны находиться изображения для баннеров на главном экране.

## Требуемые файлы:

1. **banner_friday.png** - для баннера "Скидка 10% по пятницам!"
2. **banner_express.png** - для баннера "Экспресс-уборка за 2 часа"
3. **banner_friend.png** - для баннера "Приведи друга — получи скидку!"

## Рекомендуемые размеры:

- **Ширина**: 400-800px (зависит от ширины экрана)
- **Высота**: 200px (фиксированная высота баннера)
- **Формат**: PNG или JPG
- **Соотношение сторон**: примерно 2:1 или 4:1 (ширина:высота)

## Рекомендации по дизайну:

- Изображения должны быть яркими и привлекательными
- Текст на баннерах будет отображаться поверх изображения с тенью для читаемости
- Рекомендуется использовать светлые или яркие цвета
- Избегайте слишком темных изображений, так как текст белый

## Примечание:

Если изображения не будут найдены, баннеры будут отображаться с цветным фоном (fallback).
