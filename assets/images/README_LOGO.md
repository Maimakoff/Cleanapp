# Логотип приложения

## Инструкция по добавлению логотипа

1. **Поместите файл логотипа** в эту папку (`assets/images/`)
2. **Назовите файл**: `logo.png`
3. **Рекомендуемые размеры**:
   - Минимум: 256x256 пикселей (для четкости)
   - Оптимально: 512x512 пикселей
   - Формат: PNG с прозрачным фоном (если нужно)

## Текущий логотип

Логотип должен быть изображением зеленого дома с метлой и пузырьками внутри.

## Где используется логотип

Логотип автоматически отображается в:
- Главном экране (заголовок)
- Экране авторизации
- Других местах приложения через виджет `AppLogo`

## Fallback

Если файл `logo.png` не найден, будет показана иконка-заголовка с текстом "Cleanapp".
