# Инструкция: Обновление логотипа в приложении

## Проблема
Логотип не обновился после добавления нового файла `logo.png`.

## Решение

### Вариант 1: Полная пересборка (рекомендуется)

1. **Остановите приложение** на телефоне (закройте полностью)

2. **Выполните команды в терминале:**
   ```bash
   flutter clean
   flutter pub get
   ```

3. **Пересоберите и установите приложение:**
   ```bash
   flutter run
   ```
   Или если используете Android Studio/VS Code - запустите приложение заново

### Вариант 2: Удаление и переустановка

1. **Удалите приложение** с телефона полностью
2. **Выполните:**
   ```bash
   flutter clean
   flutter pub get
   flutter run
   ```

### Вариант 3: Очистка кеша (Android)

Если используете Android:
```bash
flutter clean
cd android
./gradlew clean
cd ..
flutter pub get
flutter run
```

## Проверка

После пересборки логотип должен отображаться:
- ✅ В заголовке главного экрана
- ✅ На экране авторизации
- ✅ В других местах, где используется `AppLogo`

## Если логотип все еще не отображается

1. **Проверьте файл:**
   - Убедитесь, что файл `assets/images/logo.png` существует
   - Проверьте, что файл не поврежден (откройте его в просмотрщике изображений)

2. **Проверьте pubspec.yaml:**
   - Должна быть строка: `- assets/images/`

3. **Проверьте код:**
   - В `lib/widgets/app_logo.dart` путь должен быть: `'assets/images/logo.png'`

4. **Очистите кеш Flutter:**
   ```bash
   flutter clean
   flutter pub cache repair
   flutter pub get
   ```

## Примечание

Flutter кеширует ресурсы (assets) при сборке. Поэтому после добавления или изменения файлов изображений необходимо:
- Выполнить `flutter clean`
- Пересобрать приложение

Это нормальное поведение Flutter для оптимизации производительности.
